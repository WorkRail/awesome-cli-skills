# awesome-cli-skills

A curated registry of CLI tools and AI agent skills for WorkRail.

Each entry has two parts:
- **Binary** — Platform-specific download URLs for the CLI executable
- **Skill** — SKILL.md defining how AI agents invoke the tool

---
